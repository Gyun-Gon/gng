### Branch

`Ex. Type/#Issue.number-Title`

`master` : 최종본

`env` : 환경 설정

`feat` : 이슈 별 기능 개발

`refactor` : 리팩토링

`fix` : 오류 수정

`test` : 테스트 코드 작성

### Commit Message

`Ex. [ Type ] Title`

`env` : 환경 설정

`feat` : 기능 개발

`refactor` : 리팩토링

`fix` : 오류 수정

`chore` : Gradle 설정, 기타 작업

`test` : 테스트 코드 작성

**커밋 단위는 세부 기능 기준!!**
